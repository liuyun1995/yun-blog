正则表达式是我们经常会用到的大杀器，用好了能够减少编程时间，用不好却

#### 1 什么是语言？



| 运算          | 定义和表示 |
| ------------- | ---------- |
| L和M的并      | $ L \cup M = \left\{s|s属于L或者属于M\right\} $   |
| L和M的连接    | $ LM = \left\{st|s属于L且t属于M \right\} $           |
| L的Kleene闭包 | $ L^* = \cup_{i=0}^{\infty} L^i $ |
| L的正闭包     | $ L^* = \cup_{i=1}^{\infty} L^i $ |
